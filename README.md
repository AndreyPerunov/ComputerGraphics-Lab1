- [Threejs](https://threejs.org/) - Graphics
- [React Three Fiber](https://r3f.docs.pmnd.rs/getting-started/introduction) - Graphics
- [Tailwindcss](https://tailwindcss.com/) - Styling
- [Bun](https://bun.com/) - Runtime

## Task

A student should develop an application using the selected programming language and graphical API that:

- displays two-dimensional objects on the screen;
- demonstrates all possible objects transformations;
- allows the user to interact with the objects.
